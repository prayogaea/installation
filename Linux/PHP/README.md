# PHP Installation

### Install [PHP](https://www.php.net/downloads) on Ubuntu, Debian, Kali Linux, and Linux Mint

- Installation
```bash
sudo apt update
```
```bash
sudo apt install php8.1 libapache2-mod-php8.1
```
```bash
php -v
```

- Uninstallation
```bash
sudo apt-get purge php8.*
```
```bash
sudo apt-get autoclean
```
```bash
sudo apt-get autoremove
```
