# PHP Installation

### Install PHP on Ubuntu, Debian, Kali Linux, and Linux Mint

- Installation
```bash
sudo apt update
```
```bash
sudo apt install python3-pip
```
```bash
pip3 --version
```
```bash
pip3 list
```

- Uninstallation
```bash
sudo apt remove --auto-remove python-pip
```
```bash
sudo apt purge --auto-remove python-pip
```
