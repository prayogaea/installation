# Tor Browser Installation

### Install Chrome on Ubuntu, Debian, Kali Linux, and Linux Mint

- Installation
```bash
sudo apt update
```
```bash
sudo add-apt-repository ppa:micahflee/ppa
```
```bash
sudo apt install torbrowser-launcher
```
```bash
torbrowser-launcher
```

- Uninstallation
```bash
sudo apt remove torbrowser-launcher
```
